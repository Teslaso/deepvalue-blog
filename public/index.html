    核心领域： 因子投资、量化选股、基本面分析

    第一步：核心概要与贡献识别

    • 一句话电梯演讲 (Elevator Pitch):

      这篇文档揭示了如何将公司金融的第一性原理（投资和盈利能力）转化为可交易的投资因子，其解释力在很大程度上超越了Fama-French模型，并且它还引入了一个基于“预期”而非“历史”的前瞻性增长因子。

    第二步：分领域深度剖析与应用

    模块一：[q-因子的构建：基于经济理论的因子定义]

      1. 这个核心洞见是什么？ (What is the core insight?)

      q-因子模型（HXZ 2015）的核心是两个基于基本面的因子：投资（Investment）和盈利能力（Profitability）。

      1. 投资因子 ($R_{I/A}$):

        • 定义： 投资-资产比（I/A），即总资产的年度变化 / 滞后一年的总资产。

        • 构建： （低 I/A 投资组合的平均回报）-（高 I/A 投资组合的平均回报）。

        • 洞见： 这意味着，在其他条件相同的情况下，投资较少（即资产扩张较慢）的公司，其未来股票回报率更高。

      2. 盈利能力因子 ($R_{Roe}$):

        • 定义： 净资产收益率（Roe），即季度税前非凡利润（IBQ） / 滞后一季度的账面权益。

        • 构建： （高 Roe 投资组合的平均回报）-（低 Roe 投资组合的平均回报）。

        • 洞见： 这意味着，盈利能力更强的公司，其未来股票回报率更高。

      1. 为什么这个洞见至关重要？ (Why does it matter?)

      这为“价值”和“质量”投资提供了坚实的经济学理论基础（q-理论）。

    铜矿资本周期分析-马拉松资本

    2025-11-12

    马拉松全球投资评论 第38卷 第3号 2024年5月31日

    铜“点石成金”

    “有（些）时候，几十年（才发生）的事情会（在）几周内发生。”

    弗拉基米尔·列宁

    我们的老读者可能还记得我们在2021年12月那篇题为《回归本源》的文章。当时我们指出，由于全球金融危机（GFC）后长达十年的投资不足，铜行业即将面临供应紧缩。

    自那以后，尽管整个资本周期的大趋势和该行业面临的根本挑战几乎没有改变，但近几周来，股票市场对铜矿公司的估值却发生了翻天覆地的变化。

    铜资产的吸引力早已被产业买家看在眼里。继Glencore(Marathon持仓）早先尝试收购泰克资源（Teck Resources）之后，必和必拓（BHP，Marathon另一持仓）也紧接着试图“迎娶”英美资源集团（Anglo American）的股东们——后者是一家在拉丁美洲拥有大量铜矿敞口的同行。 这项收购提议是矿业传奇历史中规模最大、也可能（在结构上）最复杂的一笔交易。

    尽管我们很想沉浸在资本周期（按预期）上演所带来的喜悦中，但股价如此急剧的上涨，要求我们必须重新审视最初的投资逻辑，并思考在当前水平下是否仍有理由继续持有这些公司。

    我们的投资论点基于一个观察：长达十年的资本支出下降将导致供给侧日益脆弱，老旧矿山将难以维持稳定的产出水平。

    图1：铜“底” 资本支出崩塌：十亿美元（实际值）

    去年，供给紧缩的痛感变得尤为真切。第一量子矿业（First Quantum Minerals，Marathon持仓）以及随后的英美资源集团（Anglo American，未持仓）相继被迫宣布大幅下调其拉丁美洲矿山的产量预期，原因分别是政治风险和地质挑战。

    投资银行瑞银（UBS）估计，这些因素及其他问题已导致年化供给削减高达120万吨，占全球产量的5%，使市场从预期的过剩急转为短缺。

    实体商品市场需要“平衡”。在这种情况下，平衡的实现依赖于两个方面：一是从非传统来源中“挤”出额外的供给，二是通过商品价格上涨来“破坏”需求。

    “解决高价格的办法就是高价格”这句格言在过去的每个周期中都得到了印证，本轮周期无疑也将如此。但值得注意的是，鉴于近年来日益繁重的审批和环保要求，仅靠增加资本投资并不能轻易解决当前的供给短缺问题。 在上一轮周期中需要六到八年才能开发的项目，现在至少需要十年时间，并且每吨产出所需的资本投资翻了一番¹ 。 现在“绿地”（新建）铜项目（的资本开支）按最终年化产出计算，通常估计为 每吨超过2.5万美元，而在2006-2012年周期中，这个数字是 每吨1万至1.2万美元。

    我们在上一篇文章中（译注：指2021年那篇）的潜台词是：那些拥有庞大储量和优越地质条件的资产被系统性地低估了。因为随着时间的推移，那些面临更大技术和地质挑战的项目成本曲线会变得更加陡峭且上移，这将使得优质资产的价值不断增加。 资本支出通胀（Capex inflation）在整个成本曲线上的表现是非线性的，这导致了那些大型、低成本的优质储备，其经济价值会随着时间不成比例地增长。

    新矿（Greenfield，即“绿地”项目）所需的绝对资本投入远高于现有在产资产，且投产周期延长至十年以上。这使得此类“绿地”项目在时间和成本两个维度上的执行风险都大大增加。 对于考虑此类绿地投资的公司（即股东）而言，他们所要求的（最低）回报率（IRR）必须高于那些投产周期更短的时期。 其次，这带来的一个“二阶效应”是：在产资产（Brownfield，即“棕地”）相对于待开发资产（绿地）的吸引力相对增加了。

    这种系统性的低估之所以能持续存在，其真正原因在于：卖方（Sell-side）分析师的模型未能充分计算那些在（通常为30年）预测期结束后仍埋藏在地下的资源价值。 在评估“实物”资产时，这是一个尤其严重的问题，因为在行业资本投资的通胀特性下，这些资产的价值会随着时间而增值。 这一点在先前一篇《全球投资评论》（GIR）的文章中已有阐明，即《托宾Q值重生》（《全球投资评论》，2023年3月）。相比之下，我们基于资产的估值方法，不仅考虑资产当前产生的现金流，还考虑其在当前采矿计划之外所蕴含的储备价值。

    在新兴市场（EM）的模型投资组合中，我们最青睐的股票是南方铜业（Southern Copper），因为它拥有低成本、长寿命的资产基础。 该公司披露的储量是基于一个极其保守的长期铜价假设。公司目前的产量仅为其潜在年产量的约50%。这导致该股票从表面市盈率（Headline multiples）看估值很高，但如果按“企业价值/每吨资源”（EV/resource tonne）来衡量，它却是一个相对的便宜货——特别是考虑到其高质量资产基础所对应的相对较低的资本密集度。

    持有多只暴露于同一资本周期的证券³，其好处得以凸显。马拉松的“多顾问”（Multi-counsellor）模式使我们的投资组合经理能够在各自负责的区域内，根据自己的判断来表达对某一资本周期的看法。 我们的模式鼓励思想的独立性，而这些独立思考常常会汇聚到同一个理念上。

    就铜而言，这使得马拉松的投资组合在全球范围内持有了多家相关的矿业公司⁴，从而在稀释了单一股票风险的同时，强化了对资本周期的整体敞口。

    尽管第一量子（First Quantum）在巴拿马的矿山关闭对这只个股而言是痛苦的，但客户的整体投资组合却得到了很好的保护，因为供给-需求动态的收紧，推动了投资组合中其他（铜矿）公司的股价上涨⁵ 。

    我们不应忽视，最近这轮价格上涨是在美元走强、利率上升和中国建筑业处于下行周期的背景下发生的。这些因素给了那些“以需求为中心”的研究分析师们足够多的理由来回避这个行业。

    马拉松专注于供给侧，并秉持长期持有的理念，这使我们能够在更广泛的投资组合结构中“坚守”这些股票，直到资本周期最终兑现。我们能够从这轮上涨中获益，而不必去过度猜测那些非线性股价变动的确切时机。

    整体需求状况的改善将为这一资本周期理论锦上添花，但这本身并非一个必要条件。 尽管如此，已有迹象表明，铜消费特定终端市场所面临的周期性逆风可能正在减弱。

    尽管对中国房地产市场前景的焦虑仍然存在，但该领域目前仅占全球（铜）需求的约7%，不到可再生能源和电动汽车需求份额的一半。 正如我们在别处讨论过的⁶，资本支出的接力棒正传递给科技行业。这正在对那些固定资产密集型行业的产出造成越来越大的压力，而这些行业（译注：指矿业等）对其现有基础设施的投资仍然不足。

    研究表明，未来五年，数据中心可能为铜需求带来相当于一个大型铜矿（70-80万吨）的增量。 与此同时，鉴于本轮收缩阶段的深度和持续时间，许多终端需求的周期性部门（建筑、制造等）可能正接近其下行周期的尾声。

    ³ 这与大多数投资顾问（他们要求投资组合日益集中）的传统智慧背道而驰⁴ 南方铜业、第一量子矿业、自由港、嘉能可、Oz Minerals、艾芬豪矿业和必和必拓⁵ 仅在新兴市场模型投资组合中，南方铜业的收益就已远远超过持有第一量子所带来的损失。并且，作为该公司资本结构中唯一“永续工具”（译注：可能指股票或永续债）的长期持有者，我们仍可能从后者股价的复苏中受益⁶ 现实世界（《全球投资评论》，2023年9月）

    总而言之，由于供给性质发生了结构性变化，本轮铜的资本周期，其持续时间有可能比上一轮周期更长。

    虽然近期的股价飙升表明我们已经度过了这个周期的初始阶段，但通过我们评估长期持股所使用的估值视角来看，我们仍有充分理由继续持有铜矿股票。

    尽管如此，随着派对渐入高潮，我们开始（悄悄）跳得离门口更近一些。

    模块一：[贡献一：识别了“失落的十年”及其历史相似性]

    • 1. 这个核心洞见是什么？ (What is the core insight?)

      • 学术界广泛使用的法玛-弗兰奇五因子模型，在2010-2019这十年间整体失效了 。

      • 具体来说，规模 (SMB) 和价值 (HML) 因子经历了负回报 55，投资 (CMA) 因子回报接近于零 66，只有盈利 (RMW) 因子提供了正回报，但其溢价也仅为2010年以前水平的一半左右 7。

      • 将市场因子排除后，这四个因子的平均溢价在2010-2019年间为-0.28% 。

      • 然而，这并非史无前例。论文指出，2010年代的情况与1990-1999年代“惊人地相似” (remarkably similar) 。在1990年代，规模和价值因子同样为负，投资因子接近于零，导致因子组合的整体回报也为负。

    价值投资的三大支柱 #

    1. 内在价值分析 📊 #

    价值投资的核心在于寻找被低估的资产。正如沃伦·巴菲特所说：

    “价格是你付出的，价值是你得到的。”

    关键指标包括：

    • 市盈率(P/E): 衡量股价相对于每股收益的比率
    • 市净率(P/B): 股价与每股净资产的比率
    • 自由现金流: 企业真正的盈利能力

    2. 安全边际 🛡️ #

    本杰明·格雷厄姆提出的"安全边际"概念至今仍是价值投资的基石：

    # 简单的安全边际计算示例
def calculate_margin_of_safety(intrinsic_value, market_price):
    """
    计算安全边际
    :param intrinsic_value: 内在价值
    :param market_price: 市场价格
    :return: 安全边际百分比
    """
    margin = (intrinsic_value - market_price) / intrinsic_value * 100
    return f"安全边际: {margin:.2f}%"

# 示例
print(calculate_margin_of_safety(100, 70))  # 输出: 安全边际: 30.00%

    3. 长期思维 ⏰ #

    价值投资强调：

    1. 耐心等待: 市场终将反映企业的真实价值
    2. 复利效应: 时间是优质企业的朋友
    3. 避免市场噪音: 专注于基本面而非短期波动
